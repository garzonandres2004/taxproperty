# V3 Roadmap & Future Vision

## What's New in V2.1

**Version 2.1.0** — Released April 2026

V2.1 represents our demo-ready milestone with 127 Utah properties now live in the platform. This release focuses on stability, data accuracy, and user experience improvements.

### Changelog

- **Enhanced property data accuracy** — Improved address parsing and geocoding for Weber County parcels with 99.2% match rate
- **New map clustering** — Properties now group intelligently at different zoom levels, making it easier to spot opportunity zones
- **Expanded property details** — Added lot size, assessed value history, and last sale date to every property card
- **Redemption status indicators** — Visual badges now show real-time redemption risk levels (Safe / At Risk / Redeemed)
- **Performance improvements** — 40% faster load times for property searches and map interactions

## Coming Soon (V3) — Q3 2026

V3 is built and ready. These features are in final testing and will roll out throughout Q3 2026.

### 1. Salt Lake County Expansion
Multi-county architecture has been our focus since day one. V3 proves it works by adding Salt Lake County alongside Weber County — over 400 new properties and a blueprint for scaling to every county in Utah.

### 2. User Authentication
Google OAuth integration plus property ownership verification. Save searches, track parcels, receive alerts when properties you are watching hit critical milestones.

### 3. City Zoning Automation
Manual zoning lookups are painful. V3 automates this for 16 key cities: Provo, Orem, Lehi, American Fork, Pleasant Grove, Lindon, Vineyard, Highland, Alpine, Cedar Hills, North Ogden, South Ogden, West Haven, Marriott-Slaterville, Uintah, and Hooper.

### 4. Assemblage Detection
Our most requested feature. The platform now identifies adjacent parcels with the same owner — flagging potential assemblage opportunities that others miss.

### 5. Data Sync Automation
Daily automated redemption checking. No more stale data. The platform now refreshes redemption status every 24 hours from county data sources.

## Future Vision (V4+) — Q1 2027

What comes after V3 sets the multi-county foundation.

### Full Utah State Coverage
All 29 counties. From Washington County in the south to Cache County in the north. Utah will be the first fully-covered state.

### AI-Powered Risk Prediction
Machine learning models trained on historical redemption patterns, economic indicators, and property-specific factors to predict which parcels are most likely to redeem before sale.

### Comparable Sales Integration
Real-time comparable sales data (comps) integrated directly into property cards. Know what similar parcels sold for before you bid.

### Mobile Apps (iOS & Android)
Native mobile applications for on-the-go research, push notifications for new properties matching your criteria, and mobile-optimized map exploration.
